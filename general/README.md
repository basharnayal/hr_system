# نظام HR — الحضور والانصراف

نظام بسيط مكتوب بـ PHP + MySQL لإدارة الموظفين وتسجيل الحضور والانصراف.

## المتطلبات
- PHP 7.4 أو أحدث (PDO + MySQL)
- MySQL 5.7 أو MariaDB 10+
- أي استضافة فيها cPanel أو سيرفر Apache/Nginx

## التثبيت

### 1) ارفع الملفات
انسخ كل ملفات هذا المجلد إلى السيرفر (مثلاً: `public_html/hr/`).

### 2) أنشئ قاعدة البيانات
- افتح **phpMyAdmin** في cPanel
- أنشئ مستخدم وقاعدة بيانات (سمّها `hr_system` مثلاً)
- استورد ملف `install.sql` (تبويب Import) — سينشئ كل الجداول

### 3) عدّل ملف الاتصال
افتح `config.php` وعدّل:
```php
define('DB_HOST', 'localhost');
define('DB_NAME', 'hr_system');
define('DB_USER', 'اسم_المستخدم');
define('DB_PASS', 'كلمة_المرور');
```

### 4) شغّل المثبّت
افتح في المتصفح:
```
https://yourdomain.com/hr/install.php
```
- حدّد اسم الشركة
- أنشئ حساب المدير (اسم مستخدم + كلمة مرور)
- اضغط "تثبيت"

### 5) ⚠️ احذف install.php
بعد التثبيت **احذف ملف `install.php` من السيرفر** لأسباب أمنية.

---

## الاستخدام

### دخول المدير
- ادخل عبر `login.php` بحساب المدير
- لديك ٤ صفحات:
  1. **الرئيسية** — إحصائيات وحضور اليوم
  2. **الموظفون** — إضافة/حذف موظف وإنشاء حساب دخول له
  3. **سجل الحضور** — بحث بالتاريخ والموظف + تصدير CSV
  4. **الإعدادات** — تعديل اسم الشركة وكلمة مرور المدير

### إضافة الموظفين
من صفحة "الموظفون" أدخل لكل موظف:
- الاسم، الهوية، الجنس، الجوال، البريد
- اسم مستخدم وكلمة مرور للدخول

### دخول الموظف
- يدخل الموظف عبر نفس صفحة الدخول
- يرى شاشة فيها اسم الشركة وساعة + زرّي **تسجيل حضور** و **تسجيل انصراف**
- يرى آخر ٧ تسجيلات له

---

## ملاحظات أمنية
- جميع كلمات المرور مشفّرة بـ `password_hash` (bcrypt)
- جميع النماذج محمية بـ CSRF token
- جميع الاستعلامات تستخدم Prepared Statements
- **استخدم HTTPS** على السيرفر الإنتاجي
- **احذف `install.php`** بعد التثبيت

## الملفات
```
config.php             — إعدادات قاعدة البيانات
auth.php               — دوال المصادقة و CSRF
install.sql            — سكربت إنشاء الجداول
install.php            — مثبّت يُحذف بعد الاستخدام
login.php / logout.php — الدخول والخروج
index.php              — صفحة حضور الموظف
admin.php              — لوحة المدير الرئيسية
admin_employees.php    — إدارة الموظفين
admin_attendance.php   — سجل الحضور + تصدير CSV
admin_settings.php     — الإعدادات
_admin_nav.php         — شريط التنقل (include)
style.css              — التصميم
```
